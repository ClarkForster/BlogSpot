# 右边栏注释功能说明

## 功能概述

已成功实现Simon博客风格的右边栏注释功能，支持在markdown文章中添加注释，并在不同屏幕尺寸下自动切换显示方式。

## 使用方法

### 1. 段落注释（块级注释）

使用markdown的blockquote语法（`>` 开头）：

```markdown
这是正文内容。

> 这是一个段落注释，会显示在右侧边栏。

继续正文内容。
```

### 2. 行内注释

使用自定义标记 `[>]...[/]`，可以插入到段落任意位置：

```markdown
这是一段文本[>]这是行内注释内容[/]，可以继续写更多内容。
```

## 功能特性

### 响应式设计

- **大屏幕（≥ 1440px）**
  - 原文只显示上标样式的注释引用标记（编号）
  - 右侧边栏显示完整注释内容
  - 注释与原文引用标记垂直对齐
  - 布局：左侧目录 → 中间文章 → 右侧注释

- **中等屏幕（961px - 1439px）**
  - 隐藏右侧注释栏
  - 原文只显示引用标记
  - 保留左侧目录

- **小屏幕（≤ 960px）**
  - 隐藏右侧注释栏和引用标记
  - 原文显示完整的blockquote注释内容
  - 移动端友好

### 交互功能

1. **双向点击跳转**
   - 点击原文引用标记 → 滚动到右侧对应注释
   - 点击右侧注释 → 滚动到原文对应位置

2. **悬停高亮**
   - 鼠标悬停引用标记 → 对应注释高亮
   - 鼠标悬停注释 → 对应引用标记高亮

3. **自动编号**
   - 注释按出现顺序自动编号
   - 支持段落注释和行内注释混合使用

### 布局优化

- 文章内容宽度：760px（舒适阅读宽度）
- 左侧目录宽度：240px，带内边距，不紧贴边缘
- 右侧注释宽度：340px，与文章保持合理间距
- 注释跟随文章滚动，位置始终与原文对齐
- 所有元素在页面范围内，不会溢出

## 技术实现

### 核心文件

1. **`docs/.vitepress/config.ts`**
   - markdown-it插件配置
   - 支持 `>` 和 `[>]...[/]` 两种注释语法
   - 服务端渲染注释数据

2. **`docs/.vitepress/theme/components/ArticleLayout.vue`**
   - 注释侧边栏组件
   - 位置计算和对齐逻辑
   - 交互事件处理

3. **`docs/.vitepress/theme/custom.css`**
   - 注释样式定义
   - 响应式布局规则
   - 三栏布局（目录-文章-注释）

## 浏览器兼容性

- 现代浏览器（Chrome, Firefox, Safari, Edge）
- 支持CSS Grid和Flexbox
- 支持ResizeObserver API

## 性能优化

- 使用ResizeObserver监听内容变化
- 防抖处理位置重新计算
- 最小化DOM操作
- CSS硬件加速

## 已知限制

- 注释内容不支持复杂的markdown嵌套结构
- 行内注释不支持跨段落
- 需要JavaScript支持才能显示右侧边栏
